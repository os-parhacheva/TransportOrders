# 🚚 TransportOrders

## Локальный запуск проекта 

Чтобы запустить проект **TransportOrders** на локальной машине, выполните следующие шаги:

### 1. Клонирование репозитория
Склонируйте репозиторий на свой компьютер.

### 2. Очистка решения
Для решения **"TransportOrders.API"** в верхнем меню выберите:
Сброка -> **Очистить решение**.

### 3. Применение миграции
Откройте **Консоль диспетчера пакетов** и выполните для проекта **TransportOrders.Infrasrtructure** команду `Update-Database`.

### 4. Настройка запуска
1 Установите тип запуска : Несколько запускаемых проектов;

2  Выберите: 
  - TransportOrders.API -> *Запустить без отладки*;
  - TransportOrders.Presentation -> *Запустить без отладки*.

### 5. Запуск
Запустите проект.

Запуститься серверная и клиентская часть web-приложения.
